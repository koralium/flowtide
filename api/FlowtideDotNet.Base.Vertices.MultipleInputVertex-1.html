Remarks

This vertex acts as an ISourceBlock<TOutput> emitting a unified output after accepting and coordinating multiple inputs. It heavily manages stream coordination by synchronizing Watermarks, ILockingEvents (checkpoints), and FlowtideDotNet.Base.InitialDataDoneEvents across all configured input targets. This ensures the operator advances its logical state only when all discrete input branches have reached an agreed-upon synchronization point. Derived classes implement OnRecieve(int, T, long) to provide specific processing logic per target.
