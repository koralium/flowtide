Represents a precondition that must be satisfied before a SpiceDB write operation is committed.

public interface ISpiceDbPrecondition

Remarks

This interface wraps the Authzed API v1 Precondition message. Preconditions assert the existence or absence of relationships that match a given filter; if any precondition is not satisfied, the parent write is not committed. Preconditions are supplied via OptionalPreconditions. Use the static Create(SpiceDbPreconditionOperation?, ISpiceDbRelationshipFilter?) factory method to construct a new instance.
