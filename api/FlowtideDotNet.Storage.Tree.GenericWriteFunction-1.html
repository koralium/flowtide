Delegate GenericWriteFunction<V>

Namespace
FlowtideDotNet.Storage.Tree
Assembly
FlowtideDotNet.Storage.dll

A read-modify-write callback invoked for a key during a write. It receives the value supplied to the write call, the current stored value (or default when absent), and whether a value currently exists, and returns the new value together with the GenericWriteOperation to apply.

public delegate (V? result, GenericWriteOperation operation) GenericWriteFunction<V>(V? input, V? current, bool exists)

Parameters

input V

The value supplied to the write call.

current V

The current stored value, or default when the key is not present.

exists bool

True when a value already exists for the key.

Returns

(V result, GenericWriteOperation operation)

The new value and the operation to perform.

V

The value type.
