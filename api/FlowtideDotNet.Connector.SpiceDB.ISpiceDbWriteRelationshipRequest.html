Represents a request to atomically apply a set of relationship mutations to SpiceDB, with optional preconditions.

public interface ISpiceDbWriteRelationshipRequest

Remarks

This interface wraps the Authzed API v1 WriteRelationshipsRequest message. The mutations in Updates are submitted to SpiceDB via the WriteRelationships RPC. When OptionalPreconditions is non-empty, all specified preconditions must be satisfied before the write is committed; if any precondition fails the entire write is rejected. The request is also passed to BeforeWriteRequestFunc before each batch submission, allowing callers to inspect or modify the updates prior to sending. Use the static Create(IEnumerable<ISpiceDbRelationshipUpdate>?, IEnumerable<ISpiceDbPrecondition>?) factory method to construct a new instance.

IList<ISpiceDbPrecondition>

Remarks

When empty, no preconditions are evaluated and the write proceeds unconditionally. If any precondition is not satisfied, the entire write is rejected without applying any of the mutations.
