Class BitmapList

Namespace
FlowtideDotNet.Storage.DataStructures
Assembly
FlowtideDotNet.Storage.dll

A growable list of bits packed into 32-bit words and backed by unmanaged memory allocated from an IMemoryAllocator. It supports the usual list operations (add, get, set, insert and remove) as well as range and batch variants that shift large blocks of bits at once, using vectorized shifts where available. It is commonly used as a validity (null) mask for columnar data. Instances own native memory and must be disposed; Dispose() releases the buffer and returns the instance to FlowtideDotNet.Storage.DataStructures.BitmapListFactory for reuse.
