AddSpiceDbSink(IConnectorManager, string, SpiceDbSinkOptions)

Registers a SpiceDB relationship sink with the connector manager that writes relationships to tables whose names match the specified pattern.

public static IConnectorManager AddSpiceDbSink(this IConnectorManager connectorManager, string regexPattern, SpiceDbSinkOptions spiceDbSinkOptions)

Parameters

connectorManager IConnectorManager

The IConnectorManager to register the SpiceDB sink with.

regexPattern string

A regular expression pattern used to match table names declared in the stream plan. Use "*" to match all table names.

spiceDbSinkOptions SpiceDbSinkOptions

The SpiceDbSinkOptions that configure the SpiceDB gRPC channel, batch size, write hooks, watermark callbacks, and the optional deletion filter applied after the initial data load.

Returns

IConnectorManager

The same IConnectorManager instance to allow method chaining.

Remarks

The sink maps incoming rows to SpiceDB relationships using the columns resource_type, resource_id, relation, subject_type, subject_id, and subject_relation. Use "*" as regexPattern to match all table names.
