IsSnapshot

Indicates whether the checkpoint is a snapshot checkpoint or not. Snapshot checkpoints are checkpoints that contain the full state of the system at a given version, while non-snapshot checkpoints may only contain incremental changes since the last snapshot checkpoint. This information can be used to optimize checkpoint loading and recovery processes, as loading from a snapshot checkpoint can be faster than applying a series of incremental checkpoints.
