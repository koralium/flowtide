Remarks

Register this strategy via AddFailureListener(IFailureListener) or FlowtideBuilder.WithFailureListener to let a container orchestrator such as Kubernetes or Docker restart the process when the stream fails, rather than allowing the stream to attempt its built-in automatic recovery.

OnFailure(StreamFailureNotification) calls Exit(int) with the current ExitCode, which initiates a graceful shutdown: registered AppDomain.ProcessExit handlers and object finalizers are executed before the process terminates. For an immediate, non-graceful termination that skips cleanup, use KillProcessStrategy instead.
