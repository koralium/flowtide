Represents a reference to the subject portion of a SpiceDB relationship, identifying a subject by its object reference and an optional sub-relation.

public interface ISpiceDbSubjectReference

Remarks

This interface wraps the Authzed API v1 SubjectReference message. A subject reference is used as Subject to identify who holds a relation on a resource. The optional relation component enables referencing a sub-relation on the subject — for example, group:engineering#members to refer to the members of a group rather than the group itself. Use the static Create(ISpiceDbObjectReference?, string?) factory method to construct a new instance.

string OptionalRelation { get; set; }

Property Value

string

Remarks

When not set, the subject is the object identified by Object directly. When set, the subject is the set of objects reachable via the named relation on the referenced object.
