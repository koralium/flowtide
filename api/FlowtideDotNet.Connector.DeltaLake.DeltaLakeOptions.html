MaxFileSizeBytes

Gets or sets the maximum allowed file size, in bytes. Files are usually smaller than this value because of compression.

public long MaxFileSizeBytes { get; set; }

Property Value

long

Remarks

This setting is used to determine when to roll over to a new file during writes. If the current file being written exceeds this size, a new file will be created for subsequent data. Setting this value too low may result in many small files, which can degrade read performance, while setting it too high may lead to fewer but larger files, which can also impact performance and manageability. The default value is set to 100 MB.
