Serializes SubstreamEventsPayload only when a message actually crosses a silo boundary: Orleans invokes codecs at transport time, so grains on the same silo never pay for serialization. Writes the substream wire format (columnar, batch level) directly into the message and releases the events' receiver claims on the sending side; reading captures the raw payload into pooled segments without materializing the events, the communication handler deserializes lazily with the consuming operator's allocator.
