Remarks

This type is a ref struct to avoid heap allocation on each check failure and to allow its Tags field to hold a ReadOnlySpan<T>, which itself cannot appear as a field in a non-ref-struct type. It must not be stored or accessed beyond the duration of the OnCheckFailure(ref readonly CheckFailureNotification) call in which it is received; copy any required fields to local variables if they must outlive the call.

Instances are created internally by StreamNotificationReceiver each time a compiled SQL CHECK expression calls OnCheckFailure(in string, ReadOnlySpan<KeyValuePair<string, object?>>) with a failing message and diagnostic tags.
