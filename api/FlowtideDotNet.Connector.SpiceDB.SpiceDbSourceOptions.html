Remarks

Pass an instance of this class to SpiceDbConnectorManagerExtensions.AddSpiceDbSource to configure the gRPC connection, request metadata, and read consistency for the SpiceDB source. The source performs an initial load of relationships via the ReadRelationships RPC and then streams ongoing changes using the SpiceDB Watch API.

Remarks

When not set, the SpiceDB server applies its default consistency behaviour. This setting is not applied to the Watch stream used for ongoing change detection after the initial load. Use ISpiceDbConsistency factory methods to specify an explicit requirement.
