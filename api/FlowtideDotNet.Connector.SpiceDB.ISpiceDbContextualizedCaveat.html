Represents a caveat attached to a caveated relationship in SpiceDB, combining a caveat expression name with its evaluation context.

public interface ISpiceDbContextualizedCaveat

Remarks

This interface wraps the Authzed API v1 ContextualizedCaveat message. A caveated relationship is considered active only when the referenced caveat expression evaluates to true given the supplied context. Instances are assigned via OptionalCaveat. Use the static Create(string?, Struct?) factory method to construct a new instance.

Remarks

The keys must match the arguments defined for the caveat in the SpiceDB schema. These values are fixed at write time and passed to the caveat expression when the relationship is evaluated.

Parameters

The name of the caveat expression to use, as defined in the SpiceDB schema. If null, the caveat name will not be set.

context Struct

The key-value pairs to inject into the caveat expression at evaluation time. The keys must match the arguments defined for the caveat in the schema. If null, no context will be set.
