Remarks

This class provides a common structure for all watermark values, including a BatchID that is used to split up a watermark with the same value into multiple batches.

Properties

BatchID

Gets or sets the batch identifier associated with this watermark.

public long BatchID { get; set; }

Property Value

long

Remarks

The BatchID is utilized when a single watermark duration or metric needs to be processed in chunks. By incrementing this ID, the engine splits a primary watermark value into multiple, ordered sub-batches.

TypeId

Gets the unique identifier for the specific type of watermark.

public abstract int TypeId { get; }

Property Value

int

Remarks

Used internally to differentiate between various implementations of watermarks during comparisons. Ensures that operations such as equating and sorting only occur between watermarks of the exact same type.
