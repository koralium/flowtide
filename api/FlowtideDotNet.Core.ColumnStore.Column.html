Class Column

Namespace
FlowtideDotNet.Core.ColumnStore
Assembly
FlowtideDotNet.Core.dll

Represents a column of data.

It has the following state machine: Start -> Null column Add null value -> Null column Add value -> Value column (single type) Add different type -> Union column
