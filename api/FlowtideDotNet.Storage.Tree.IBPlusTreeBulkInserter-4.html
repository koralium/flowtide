Interface IBPlusTreeBulkInserter<K, V, TKeyContainer, TValueContainer>

Namespace
FlowtideDotNet.Storage.Tree
Assembly
FlowtideDotNet.Storage.dll

Applies a whole batch of rows to a B+ tree in a single pass. The batch is sorted by key and grouped by the leaf each row lands in, so each affected leaf is loaded once and any required node splits and merges are handled together. This is more efficient than upserting rows one at a time when writing a large batch. Each row's effect (insert, update, delete or no change) is decided by an IRowMutator<K, V>.

SortAndGetIndices(K[], int)

Sorts the first keyLength keys and returns their indices in sorted order, without reordering the input array. The result can be passed to the overload of ApplyBatch<TMutator>(K[], V[], int, TMutator, int) that takes precomputed indices to avoid sorting again. The returned array is an internal buffer that is reused between calls.
