Class NormalizationRelation

Namespace
FlowtideDotNet.Substrait.Relations
Assembly
FlowtideDotNet.Substrait.dll

A relation that normalizes the input based on a key column. Any update on that key will replace the current value with the new value. A delete will delete the key even though there have been multiple inserts before with different values but the same key.
