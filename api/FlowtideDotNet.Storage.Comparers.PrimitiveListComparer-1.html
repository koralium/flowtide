Namespace
FlowtideDotNet.Storage.Comparers

A B+ tree comparer for keys that are unmanaged, naturally comparable values stored in a PrimitiveListKeyContainer<T>. It orders keys using their IComparable<T> implementation and locates them with a binary search over the container's contiguous buffer.

public class PrimitiveListComparer<T> : IBplusTreeComparer<T, PrimitiveListKeyContainer<T>> where T : unmanaged, IComparable<T>
