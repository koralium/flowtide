Struct SerializableObject

Namespace
FlowtideDotNet.Storage
Assembly

A value handed to a file cache or persistent storage session to be written. It carries the data in one of two forms so callers do not have to serialize ahead of time: either a live ICacheObject together with the IStateSerializer that knows how to serialize it, or a block of bytes that has already been serialized. The bytes form is used when data is copied straight from the file cache to persistent storage, or for already serialized blobs such as metadata, where re-serialization is not needed.
