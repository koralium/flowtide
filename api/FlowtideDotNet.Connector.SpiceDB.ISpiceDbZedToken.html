Represents a ZedToken that encodes causality metadata for SpiceDB read and write operations.

public interface ISpiceDbZedToken

Remarks

This interface wraps the Authzed API v1 ZedToken message. ZedTokens are returned by SpiceDB after each write operation and can be stored and supplied back in subsequent read operations to establish causality guarantees. They are used as AtLeastAsFresh to ensure reads reflect at least a known write, or as AtExactSnapshot to read at a precise revision. The raw token string is also surfaced by OnWatermarkFunc after each batch is committed. Use the static Create(string) factory method to construct a new instance.
