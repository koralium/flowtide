Namespace FlowtideDotNet.Base.Vertices

Classes

EgressVertex<T>

Abstract base class for stream egress vertices (sinks) that consume stream events from the dataflow pipeline.

FixedPointVertex<T>

Abstract base class for stream vertices that implement fixed-point iteration over a cyclic dataflow graph.

IngressOutput<T>

Provides an output for an IngressVertex<TData> to push data and watermarks into the stream.

IngressVertex<TData>

Base class for stream ingress vertices (sources) that produce stream events into the dataflow pipeline.

MultipleInputTargetHolder

Holder that represents one input slot of a MultipleInputVertex<T>, implementing ITargetBlock<TInput> so that upstream sources can be linked directly to it.

MultipleInputVertex<T>

Vertex capable of receiving and processing input from multiple distinct upstream sources concurrently.

PartitionVertex<T>

Vertex responsible for partitioning an input stream into multiple target streams.

PartitionedOutputVertex<T>

Represents a specialized vertex that gathers data from multiple upstream partitions and merges them back into a single continuous stream.

UnaryVertex<T>

Base class for stream vertices that have a single input stream and one or more output streams.

Interfaces

IStreamEgressVertex

Represents an egress node (sink operator) within the Flowtide dataflow stream.

IStreamIngressVertex

Represents an ingress node (source operator) within the Flowtide dataflow stream.

IStreamVertex

Represents a foundational node (operator or vertex) within the Flowtide dataflow stream. Defines lifecycle methods (Setup, Initialize, CreateBlock, Link), operational hooks (Pause, Resume, Compact), and core metadata properties.
