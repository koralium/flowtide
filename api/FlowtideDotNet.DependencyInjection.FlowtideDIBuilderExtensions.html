AddOpenLineageHttp(IFlowtideDIBuilder, Action<OpenLineageHttpOptions>)

Adds OpenLineage HTTP reporting to the Flowtide stream, enabling lineage events to be sent to an OpenLineage-compatible endpoint.

public static IFlowtideDIBuilder AddOpenLineageHttp(this IFlowtideDIBuilder builder, Action<OpenLineageHttpOptions> configure)

Parameters

builder IFlowtideDIBuilder

The IFlowtideDIBuilder to configure.

configure Action<OpenLineageHttpOptions>

A delegate used to configure the OpenLineageHttpOptions.

Returns

IFlowtideDIBuilder

The same IFlowtideDIBuilder instance for chaining.

Remarks

The configure delegate is invoked immediately to populate an OpenLineageHttpOptions instance. At a minimum, Url must be set; otherwise an ArgumentException is thrown.

Example usage: 

builder.AddOpenLineageHttp(opt =>
{
    opt.Url = "http://localhost:8080/openapi/openlineage/api/v1/lineage";
    opt.IncludeSchema = true;
});
