Interface IRowMutator<K, V>

Namespace
FlowtideDotNet.Storage.Tree
Assembly
FlowtideDotNet.Storage.dll

Decides, per row, how a bulk insert applies an incoming row to a B+ tree. The bulk inserter calls Process(K, bool, in V, ref V, int) for each row to merge the incoming value with any existing value and choose the operation to perform, and GetSizePrefixSum(K[], ReadOnlySpan<int>, Span<int>) to estimate the serialized size of a run of rows for split planning.

GetSizePrefixSum(K[], ReadOnlySpan<int>, Span<int>)

Accumulates, into sizes, the running prefix sum of the serialized byte size of the rows at the given indices. The bulk inserter uses this to decide where a page should be split.

Process(K, bool, in V, ref V, int)

Processes a single incoming row and returns the operation to apply for it. When the key already exists the mutator can merge existingData into incomingData before the result is stored.
