CustomBulkCopyDestinationTable

If set, the sink writes data to the custom table and does not trigger any merge into to another table. For custom merge into logic with custom destination table you can use OnDataUploaded event to run any sql code. When using a custom destination table, the metadata if it's an upsert or delete is not sent, that can be added manually using ModifyRow.
