Represents a SpiceDB relationship that specifies how a resource relates to a subject.

public interface ISpiceDbRelationship

Remarks

This interface wraps the Authzed API v1 Relationship message. Relationships form the data for the graph over which all permissions questions are answered. They are carried as the Relationship field when writing to SpiceDB via ISpiceDbWriteRelationshipRequest. Use the static Create(ISpiceDbObjectReference?, string?, ISpiceDbSubjectReference?, ISpiceDbContextualizedCaveat?) factory method to construct a new instance.

When null, the relationship is unconditional. When set, the caveat expression named by CaveatName is evaluated with the provided context at permission-check time.

Parameters

resource ISpiceDbObjectReference

The ISpiceDbObjectReference identifying the resource to which the subject is related. If null, the resource will not be set.

relation string

The name of the relation between the resource and the subject (for example, "reader"). If null, the relation will not be set.

subject ISpiceDbSubjectReference

The ISpiceDbSubjectReference identifying the subject that holds the relation on the resource. If null, the subject will not be set.

optionalCaveat ISpiceDbContextualizedCaveat

An optional ISpiceDbContextualizedCaveat that conditionalizes the relationship. If null, no caveat will be applied and the relationship is unconditional.
