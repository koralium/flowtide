Interface IAppendTreeIterator<K, V, TKeyContainer>

Namespace
FlowtideDotNet.Storage.AppendTree
Assembly

A forward, read-only cursor over the entries of an IAppendTree<K, V, TKeyContainer, TValueContainer>. Seek(in K, in IBplusTreeComparer<K, TKeyContainer>?) must be called to position the cursor before enumerating; iterating without seeking first throws. Once positioned, enumerate the iterator to walk entries in ascending key order from the seek position onward.

public interface IAppendTreeIterator<K, V, TKeyContainer> : IAsyncEnumerable<KeyValuePair<K, V>>, IDisposable where TKeyContainer : IKeyContainer<K>
