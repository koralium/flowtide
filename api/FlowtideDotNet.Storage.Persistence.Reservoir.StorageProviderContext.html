Remarks

A StorageProviderContext is constructed by the storage system and passed to InitializeAsync(StorageProviderContext, CancellationToken) before the provider is used. Storage providers use the values it exposes to derive stream-specific storage paths and to allocate any native or pooled memory buffers they require at runtime.

The logical name of the stream that owns the storage provider.

The version string used to isolate this stream's data within the storage backend. This is typically derived from the stream's plan hash or a semantic version identifier.

StreamVersion

public string StreamVersion { get; }

Property Value

string

Remarks

This value is typically a plan hash or a semantic version string derived from StreamVersionInformation. Storage providers use it to scope stored data to a particular stream version, enabling multiple incompatible plan versions to coexist in the same backend without interfering with each other.
