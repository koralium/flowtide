BeforeDeleteFunc

Function that is run before each delete of a tuple. Makes it possible to send true/false back where true executes the delete and false does not. This method can be used to store tuples in an external store that this integration has handled. Can be beneficial if one wants to remove all tuples added by the integration.

This method can be called multiple times in parallell.
