Represents a reference to a specific object in the SpiceDB permission system, identified by its type and unique identifier.

public interface ISpiceDbObjectReference

Remarks

This interface wraps the Authzed API v1 ObjectReference message. Object references identify the resource side of a relationship via Resource, and form the object component of a subject reference via Object. Use the static Create(string?, string?) factory method to construct a new instance.

Remarks

Supports the wildcard value "*" to match all objects of the given type when used in relationship filters.
