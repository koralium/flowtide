Remarks

This vertex acts as an IPropagatorBlock<TInput, TOutput> in the TPL Dataflow pipeline. It handles common stream lifecycle events: restoring state, processing messages, forwarding watermarks, managing backpressure, and handling locking/checkpoint sequences. Derived classes must implement custom data processing logic using OnRecieve(T, long)
