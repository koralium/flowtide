Remarks

Both the enumerable and its inner enumerator are singletons. Instance is a shared static instance of the enumerable, and GetAsyncEnumerator(CancellationToken) always returns the same static Enumerator instance. This means the type produces no heap allocations at all, whether accessed via Instance or by calling GetAsyncEnumerator(CancellationToken).

It is used throughout the dataflow vertex pipeline wherever a method returning IAsyncEnumerable<T> must produce no output, for example when a checkpoint or watermark handler chooses to suppress output without starting an iterator state machine. For the single-element counterpart see SingleAsyncEnumerable<T>.
