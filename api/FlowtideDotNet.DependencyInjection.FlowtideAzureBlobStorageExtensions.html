AddAzureBlobStorage(IFlowtideStorageBuilder, Action<FlowtideAzureBlobOptions>)

Configures Azure Blob Storage as the persistent storage provider for Flowtide.

public static IReservoirBuilder AddAzureBlobStorage(this IFlowtideStorageBuilder storageBuilder, Action<FlowtideAzureBlobOptions> options)

Parameters

storageBuilder IFlowtideStorageBuilder

The storage builder to configure with Azure Blob Storage support.

options Action<FlowtideAzureBlobOptions>

A delegate to configure the options for Azure Blob Storage integration. Cannot be null.

Returns

IReservoirBuilder

The same storage builder instance, configured to use Azure Blob Storage for persistent storage.

Remarks

Use this method to enable Flowtide to store data in Azure Blob Storage, with optional local disk caching and customizable storage options.

AddAzureBlobStorage(IFlowtideStorageBuilder, string, string, string?, string?)

Adds Azure Blob Storage as a storage provider using the provided connection string, container name, and optional directory path.

public static IReservoirBuilder AddAzureBlobStorage(this IFlowtideStorageBuilder storageBuilder, string connectionString, string containerName, string? directoryPath = null, string? localCacheDirectory = null)

Parameters

storageBuilder IFlowtideStorageBuilder

The storage builder to which Azure Blob Storage will be added.

connectionString string

The connection string used to authenticate and connect to the Azure Blob Storage account. Cannot be null or empty.

containerName string

The name of the Azure Blob Storage container to use. Cannot be null or empty.

directoryPath string

An optional directory path within the container where data will be stored. If not specified, the root of the container is used.

localCacheDirectory string

An optional local directory path for caching data. If not specified, a default temporary directory will be used for caching.

Returns

IReservoirBuilder

The same storage builder instance with Azure Blob Storage configured as a storage provider.
